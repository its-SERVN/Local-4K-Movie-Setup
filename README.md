# 🎬 بيئة مشاهدة محلية (4K Local Setup)

## 4️⃣ المشاهدة والتحكم (VLC)
آخر خطوة، نشغل الفيلم ونركب الترجمة، ونتحكم بالتوقيت لو فيه تأخير.

![تشغيل الفيلم](./vlc-(4)/1.png)
![تركيب الترجمة](./vlc-(4)/2.png)

### ⌨️ اختصارات تهمك في VLC:
* **حرف G:** تقديم الترجمة (لو كانت متأخرة عن الكلام).
* **حرف H:** تأخير الترجمة (لو كانت سابقة الكلام).
