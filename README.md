# Local movie Setup

تجهيز بيئة كاملة على جهازك الشخصي لتحميل ومشاهدة الأفلام والترجمة بجميع أنواع الجودات. 

* ✅ **بدون إعلانات:** مابتطلع لك نوافذ منبثقة أو إعلانات مزعجة.
* ✅ **بدون تعليق:** الفيلم بيتحمل على الهاردسك، يعني ما يعتمد على سرعة الشبكة.
* ✅ **أمان:** كل شي محلي بدون مواقع أو روابط مشبوهة.
* ✅ **جودة عالية:** يمديك تحمل أي جودة حتى 4K مع جودة الصوت الأصلي (DTS/Dolby).
* ✅ **تحكم كامل:** يمديك تتحكم في وقت ظهور واختفاء الترجمة، تحكم فالصوت، والألوان، السطوع.
* ✅ **سرعة التحميل:** فيلم الـ Full HD ياخذ تقريباً **5 دقائق** تحميل ويكون جاهز.

---

## ⚠️ العيوب

* 🛠️ **التحميل يدوي:** الطريقة تتطلب خطوات يدوية للبحث والتحميل ودمج الترجمة.

---

## 🛠️ طريقة التشغيل (Deployment Modes)

في طريقتين لتجهيز البيئة:

1. **الطريقة اليدوية (Manual):** شرح مفصل بالصور لكل خطوة (موجود هنا).
2. **الطريقة التلقائية (Automated - قريباً):** سكربت برمجي يحمل البيئة مع إمكانية البحث عن الفيلم بدون الدخول للمواقع.
